## Descripción
Aplicación web para gestión de productos de una pastelería.

Permite:
- Ver productos
- Añadir productos
- Gestionar contenido desde panel admin

## Acceso

Usuario admin:
Solicitar al propietario del proyecto

Cualquier otro usuario:
Puede registrarse desde la aplicación

## Tecnologías
- PHP
- MySQL
- HTML / CSS

## Instalación

1. Importar la base de datos (db.sql)
2. Configurar conexión en /includes/config.php
3. Ejecutar en servidor local (XAMPP, etc.)
